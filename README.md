## 🚀 Getting Started (Run Locally)

```bash
# Clone the repo
git clone https://github.com/sudarshanms07/Personal_Portfolio.git

# Navigate to project
cd Personal_Portfolio

# Install dependencies
npm install

# Run development server
npm run dev
```
